## Détail des modifications

### `images/mosaic.svg` — Bandeau de la navbar

**Problème** : à chaque répétition du tile SVG (400 px), une jointure verticale visible apparaissait car les couleurs des triangles de coin gauche ne correspondaient pas aux triangles de coin droit.

**Correction** : les deux triangles de coin gauche ont été recolorés pour correspondre exactement aux coins droits :

| Triangle | Avant | Après |
|----------|-------|-------|
| Coin haut-gauche `(0,0 20,0 0,30)` | `#E3F2FD` | `#B2EBF2` (= coin haut-droit) |
| Coin bas-gauche `(0,30 0,60 20,60)` | `#E8F5E9` | `#E1BEE7` (= coin bas-droit) |

Note : les lignes horizontales à y=30 (artefact d'anti-aliasing SVG entre les polygones) sont inhérentes à l'architecture losange avec triangles de couleurs distinctes ; elles ne peuvent être éliminées sans changer le design.

### `styles.css` — Feuille de style

**Couleur des liens** : remplacement du vert par défaut du thème Flatly (`#18BC9C`) par un mauve harmonieux avec la palette du bandeau (qui contient `#E1BEE7`) :

```css
a {
  color: #8369AB;
}
a:hover {
  color: #5e4a7a;
}
```

**Couleur des liens TOC sidebar** : application de la même couleur mauve aux liens de la table des matières flottante (colonne de droite) :

```css
#TOC .nav-link          { color: #8369AB; }
#TOC .nav-link:hover    { color: #5e4a7a; }
#TOC .nav-link.active   { color: #8369AB; border-left-color: #8369AB; }
```

Choix retenu (option 1 sur 3 proposées) : mauve `#8369AB`, dérivé de `#E1BEE7` (rose-lavande du bandeau), ni bleu ni vert, harmonieux avec la palette générale.
