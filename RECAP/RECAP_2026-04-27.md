### 2. Publier sur GitHub Pages

#### Installer Git
Télécharger sur https://git-scm.com/download/win et installer avec les options par défaut.

#### Créer le dépôt GitHub
1. Aller sur https://github.com et créer un compte si nécessaire
2. Créer un **nouveau dépôt** nommé exactement `username.github.io` (remplacer `username` par votre pseudo)
3. Laisser le dépôt **public**, ne pas initialiser avec un README

#### Générer le site
Dans PowerShell, depuis le dossier `site-perso` :
```bash
quarto render
```
Cela crée le dossier `docs/` que GitHub Pages va servir.

#### Initialiser Git et pousser
```bash
cd C:\Users\jlega\Documents\site-perso
git init
git add .
git commit -m "Initial site"
git remote add origin https://github.com/username/username.github.io.git
git push -u origin main
```

#### Activer GitHub Pages
1. Sur GitHub, aller dans **Settings → Pages**
2. Source : `Deploy from a branch`
3. Branch : `main` / Folder : `/docs`
4. Cliquer **Save**

Le site sera en ligne à `https://username.github.io` dans quelques minutes.

#### Mettre à jour le site par la suite
À chaque modification :
```bash
quarto render
git add .
git commit -m "description de la mise à jour"
git push
```
